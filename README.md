# Student-Management-System
My first real world project in OOP in C++
Overview

This project is a console-based Student Management System developed in C++ using Object-Oriented Programming (OOP) principles.
It was created to practice core C++ concepts while implementing a real-world academic record management system.

Features
Dynamic handling of multiple students and subjects
Percentage calculation based on subject marks
Grade assignment using defined criteria
Sorting students by percentage (descending order)
Search functionality by student name
Handles duplicate names using roll number
Formatted output for better readability

OOP Concepts Used
Classes and Objects
Constructors and Destructors
Dynamic Memory Allocation (new / delete)
Pointers and Double Pointers
Friend Functions
Encapsulation
